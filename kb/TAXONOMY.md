# Taxonomy

## Category Tags

- `web`: GitHub.com web experiences
- `vscode`: Visual Studio Code and extensions
- `visual_studio`: Microsoft Visual Studio IDE
- `jetbrains`: JetBrains IDE plugin surface
- `xcode`: Xcode extension surface
- `cli`: Command-line tools and shell-first interfaces
- `enterprise`: Enterprise rollout, policy, and administration
- `security`: Safety, permissioning, and governance controls
- `general`: Cross-surface/general guidance

## Source Types

- `docs`: Official documentation pages
- `changelog`: Chronological product change stream
- `release-notes`: Versioned release notes
- `repo`: Source repository
- `marketplace`: Marketplace listing/version pages
- `rss`: RSS feed endpoint
- `atom`: Atom feed endpoint
- `api`: Structured API endpoint
