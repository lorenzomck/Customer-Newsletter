# Editorial Intelligence

## Theme Detection Rules

- If 3+ items share a common thread (e.g., "agent capabilities"), consider a lead section
- GA announcements always lead over PREVIEW
- Security/compliance items get priority weighting

## Selection Weights

- Enterprise governance: 3.0x
- Security/compliance: 2.5x
- GA status: 2.0x
- Novelty: 2.0x
- IDE parity: 1.5x
- Standard features: 1.0x

## Expansion Triggers

- Major GA launch → dedicated subsection
- 3+ related items → bundled narrative
- Competitive positioning signal → expanded context

## Formatting Patterns

- Bold product names at first mention
- GA/PREVIEW labels after feature name
- Enterprise "so what" in every bullet
- Link priority: Announcement > Docs > Release Notes > Changelog
