# Source Intelligence Meta-Analysis

## Key Findings

- GitHub Changelog is the richest source (80% feature survival rate)
- VS Code release notes contain 10-20x more detail than changelog entries
- JetBrains has highest survival rate (77%) — always bundled into IDE Parity
- Xcode has lowest survival (37%) — only cross-IDE features survive
- The pipeline does NOT cut items — real value is in ADDITIONS, BUNDLING, and EXPANSION
